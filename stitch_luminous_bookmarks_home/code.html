Understanding LLM Context Windows and Attention Mechanisms

Source Snippet

The architecture of modern large language models relies heavily on the attention mechanism. As the context window grows, the computational complexity scales quadratically with the input length. 

def scaled_dot_product_attention(q, k, v, mask=None):
    """Calculate the attention weights."""
    matmul_qk = tf.matmul(q, k, transpose_b=True)
    dk = tf.cast(tf.shape(k)[-1], tf.float32)
    scaled_attention_logits = matmul_qk / tf.math.sqrt(dk)

    if mask is not None:
        scaled_attention_logits += (mask * -1e9)

    attention_weights = tf.nn.softmax(scaled_attention_logits, axis=-1)
    output = tf.matmul(attention_weights, v)
    return output, attention_weights

By normalizing the dot product, we prevent the softmax function from reaching regions with extremely small gradients, which ensures stable backpropagation during training cycles.
